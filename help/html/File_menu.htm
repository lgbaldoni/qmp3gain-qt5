File menu

The File menu has the following items

Load Analysis Results This loads previously saved MP3Gain analysis results.
Save Analysis Results This saves all analysis results (Volume, Track Gain, Album Gain, etc.) currently shown in the File List.
Add Files (Ctrl-F) This allows you to add files to be corrected. Added files appear in the File List.
Add Folder (Ctrl-D) This allows you to add an entire folder instead of having to manually select all the mp3 files in a folder. Added folders appear in the File List.
Select All Files (Ctrl-A) This selects every file in the File List.
Select No Files (Ctrl-N) This unselects every file in the File List.
Invert selection (Ctrl-V) This inverts or toggles the selection status of every file in the File List. If a file is selected, it becomes unselected. If a file is unselected, it becomes selected.
Clear selected file(s) This will remove all selected files from the File List.
Clear All files This will remove all files from the File List.
Exit This quits MP3Gain.
