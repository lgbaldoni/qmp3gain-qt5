Log Panel

Log Panel is an optional section inside the user interface of the QMP3Gain application. It contains a read only, scrollable text area for the log output, some checbox input elements for (un)selecting the type of the output to be displayed, and a button for erasing the content of the log from the text area.

There are 5 types of possible log output, all of them have different colors. The last 2 are not visualized at default, but they can be enabled in the Options menu.

Error Error messages emerged during the operation of the application
Analysis Analysis results made on the MP3 files
Change Volume gain modifications made on the MP3 files
Back end (optional) Application communication with the MP3Gain backend
Trace (optional) Application trace information escpecially for possible bug hunting

Log Panel is closeable but anytime it can be opened again in the Options/Logs/Panel menu. Moreover this panel is easily detachable from the main window, its window can be resized, closed or docked to the bottom part of the main window again.

If the Log Panel is closed then no log output is written to the panel at all, but you may also use Option/Logs/File menu to send the output to a file instead of the screen. Moreover both output channels can be used in parallel.
