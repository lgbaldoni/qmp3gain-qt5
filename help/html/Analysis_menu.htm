Analysis menu

The Analysis menu has the following items, which are also available by right-clicking in the file list (note):

Track Analysis (Ctrl-T) This will analyze each file individually and then list details of the file. See the File List section for information about the columns.
Album Analysis (Ctrl-U) This will analyze each file in the Album as part of a whole and then list details about the files. See the File List section for information about the columns.
Max No-clip Analysis (Ctrl-M) (Only visible when Maximizing features are enabled)
If you are only interested in maximizing the volumes of your files, this will find the peak amplitudes of each of the mp3s in the list without spending any extra time doing Replay Gain calculations to determine how loud the file actually sounds to the human ear.
Clear Analysis This will clear the analysis results in the File List. The files will remain in the list, but will have their column info deleted.

(note) - When you use the right-click popup menu, MP3Gain will only perform the chosen function on files that are currently selected in the File List.
