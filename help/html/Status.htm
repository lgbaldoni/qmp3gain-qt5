Status

The two Status bars and the Status line at the bottom of the QMP3Gain interface are designed to give you an idea about what QMP3Gain is currently doing and how far along a file or batch is during Analysis or Gain operations.

 

If a single file is being worked on, then both the File progress and Total progress bars will be updated and they always show the same values. The Status line will show what QMP3Gain is currently doing.

 

If several files are being worked on, then both the File progress and Total progress bars will be updated. The Status line will show what QMP3Gain is currently doing. In the example below, the current file is 29% finished, while the entire batch of files being processed is 82% finished.
