Logs

You can have QMP3Gain record its operations to either a log file or to the screen named Log Panel. :

Here are the options in the "Options - Logs" submenu:

  • Log file...: Selecting this item will launch the Log file window, where the output log file can be defined.
  • Timestamp: If checked, then each message sent to the log contains a timestamp prefix. Both the file and screen logs can be checked differently.
  • Output type: Log output can be diversed by different types which are described in Log Panel.
  • Panel: Opens the Log Panel and the log are written to there if at least one of the Options/Logs/Output Types is checked.
  • Open automatically panel: Opens the Log Panel if QMP3Gain runs into any error. Unless this option is checked, then various pop up messages inform about the errors.
